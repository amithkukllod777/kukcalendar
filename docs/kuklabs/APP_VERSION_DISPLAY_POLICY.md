# Kuklabs App Version Display Policy

**Status: Mandatory**

## Never show version on

- Login
- Sign Up
- OTP verification
- Forgot Password
- Reset Password
- Welcome/onboarding
- Splash

The last authentication-screen element is:

```text
Powered by Kuklabs
```

## Show version only at the end

```text
Profile → About this app → bottom-most muted footer
```

Approved format:

```text
Version 2.4.1 (Build 24107)
```

Use muted centre-aligned text. Do not use the product accent colour. Do not expose release channel, environment or commit hash to normal production users.
